# Kalshi AI

An AI-powered football prediction platform that analyzes upcoming matches,
estimates probabilities using statistical models and machine learning,
compares them against Kalshi market prices,
and recommends the highest expected-value betting opportunities.

---

## Tech Stack

### Backend
- Python
- FastAPI
- SQLAlchemy
- PostgreSQL

### Frontend
- Next.js
- React
- TypeScript

### Machine Learning
- Scikit-Learn
- XGBoost
- LightGBM
- SciPy

### Database
- PostgreSQL

### Deployment
- Docker
- Vercel
- Render
